# Muhammad Ziyam Khan — Personal Portfolio

## Overview

This project is my official personal portfolio website, designed to showcase my journey as a Founder, Front-End Developer, UI/UX Designer, Technology Educator, and Product Builder.

The website follows a modern design philosophy inspired by leading digital experiences, combining elegant visuals with smooth interactions to create an engaging user experience. It highlights my professional work, achievements, projects, speaking engagements, and the mission behind NanoCoders.
